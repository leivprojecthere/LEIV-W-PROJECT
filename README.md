# LEIV‑W Project

## 📖 Philosophy
LEIV‑W is built on a simple principle:  
**We don’t care who you are, where you’re from, your age, gender, or social rank. We care about the intel you’ve got.**

- **No hierarchy** → President, VIP, or anonymous whistleblower, all are equal.  
- **Meritocracy of truth** → Intel is judged only on credibility, not identity.  
- **Whistleblowing accepted** → Government surveillance, censorship, or abuse can be exposed here.  
- **Strict rules** → Encryption, metadata removal, and OPSEC are mandatory for sensitive cases.  

---

## 📊 Sensitivity Scale Overview
Intel submissions are classified by sensitivity (1–10).  
The higher the score, the stricter the rules.  

- **Tier 1–3 (Low)** → Public OSINT, recon.  
- **Tier 4–6 (Medium)** → Internal leaks, corporate surveillance.  
- **Tier 7–8 (High)** → Government surveillance, censorship.  
- **Tier 9–10 (Critical)** → State secrets, military surveillance, life‑threatening intel.  

⚡ **Important:** For Tier 9–10, photos are recommended — but ONLY if stripped of all metadata (EXIF, GPS, author tags). Raw metadata will expose you.
